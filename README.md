# 🚀 Modern Landing Page

A modern, responsive landing page built using **HTML, Tailwind CSS, and JavaScript**.
Designed for startups and product websites with a clean UI and conversion-focused layout.

---

## ✨ Features

- 📱 Fully responsive (mobile-first design)
- 🎯 Conversion-focused layout
- 🧭 Sticky navbar with mobile menu
- ⚡ Smooth UI and interactions
- 🧩 Modular and easy-to-customize sections

### Sections Included

- Hero section with CTA
- Features section with cards
- About section
- Testimonials section
- Pricing section
- FAQ (accordion)
- Contact form
- Footer

---

## 🛠️ Tech Stack

- HTML5
- Tailwind CSS
- JavaScript

## 🚀 Getting Started

### 1. Clone the repository

```
git clone <repository-url>
```

### 2. Open the project

```
cd modern-landing-page
```

### 3. Run the project

Simply open:

```
src/index.html
```

in your browser.

---

## 🎯 Usage

- Edit content in `src/index.html`
- Customize styles in `src/styles/main.css`
- Add functionality in `src/scripts/main.js`
